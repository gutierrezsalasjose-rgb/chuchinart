# Chuchin Art - Sitio web inicial

Archivos incluidos:

- index.html
- style.css
- script.js
- carpeta images/

## Cómo ver la página en tu computador

Abre `index.html` con doble clic en tu navegador.

## Cómo subir a Hostinger

1. Entra a Hostinger / hPanel.
2. Abre File Manager.
3. Entra a la carpeta `public_html`.
4. Borra archivos de prueba si existen.
5. Sube `index.html`, `style.css`, `script.js` y la carpeta `images`.
6. Abre https://chuchinart.com
